RS WCL : Documentation d'utilisation

Bienvenue

Cette page décrit le fonctionnement de la RS WCL.

Des exemples de codes HTML et javascript vous y seront présentés pour chacun des composants qu'elle implémente, afin de vous permettre une prise en main rapide.

Vous pourrez également télécharger la dernière version de la librairie via le menu.

La présente page s'appuie sur la RS WCL. En effet, le menu à gauche de l'écran est une instance du composant rs-side-menu. Comme tous les composants de la librairie, son rendu peut être modifié avec CSS, afin de respecter une charte graphique donnée.
