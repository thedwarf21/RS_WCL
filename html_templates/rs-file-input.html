Description

Ce composant est un champ de saisie de type fichier sous la forme d'une vignette de visualisation. Il a été conçu pour les images uniquement. La valeur obtenue via ce composant est le code base64 de l'image sélectionnée.

Démo

Utilisation par template HTML

HTML

Utilisation en javascript pur

js

Options de paramétrage

Ce composant propose diverses options de paramétrage via les attributs de la balise :

  • onimagechange : expression évaluée lorsque l'utilisateur sélectionne une image.
  • ratio : permet de forcer un ratio, ce qui peut être pratique si votre design en dépend.
  • max_size : permet de limiter la taille des images sélectionnées. Un message d'erreur apparaît en cas de sélection d'une image trop volumineuse.
  • class : permet d'affiner le rendu graphique.

Conseils d'utilisation

Comme toujours lorsque cette possibilité est offerte, il est recommandé d'utiliser ce composant en tant que balise HTML.

En effet, comme vous pouvez le constater, bien que ce soit techniquement faisable, créer le composant en javascript pur est inutilement lourd et moins lisible.
