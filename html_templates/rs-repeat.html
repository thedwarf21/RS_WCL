Description

Ce composant sert à générer du contenu HTML à partir des éléments d'une liste ou des propriété d'un objet, en s'appuyant sur le pattern HTML contenu par la balise.

Le choix du type de parcours est défini par l'opérateur utilisé :

  • of : parcourt des éléments d'une liste
  • in : parcourt des propriétés d'un objet

Le binding intégré au composant, permet une synchronisation en temps réel entre l'interface et la liste parcourrue.

Il est possible d'ajouter un fichier CSS au shadow DOM du composant afin de gérer la mise en page au sein de la boucle de manière classique (et plus propre) via l'attribut css de la balise. Afin d'éviter des ralentissements dûs à un trop lourd volume de code CSS à charger (surtout dans le cas de boucles imbriqués susceptibles d'engendrer un grand nombre d'itérations), il est préférable de choisir un fichier CSS le plus léger et synthétique possible.
