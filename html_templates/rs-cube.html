Description

Ce composant permet d'afficher un cube animé paramétrable, non seulement du point de vue du contenu des faces qui le composent, mais également du point de vue de l'ordre de présentation de ses faces et du temps d'affichage de chacune avant de passer à la suivante.

Démo

Les faces du cube s'affichent dans l'ordre choisi

Paramétrage du rendu graphique

Ce composant utilise des variables CSS dans le rendu de son affichage. Cette fonctionnalité a été intégrée à titre expérimental pour le moment, mais les autres commposants devraient en bénéficier prochainement.

Pour paramétrer le rendu, il vous suffira de déclarer et d'initialiser à votre convenance, dans le CSS de votre page, les variables CSS correspondant à ce que vous souhaitez personnaliser :

  • --scene-bg : fond appliqué à la scène d'animation 3D.
  • --cube-face-bg : fond appliqué à chacune des faces du cube.
  • --cube-text-color : couleur appliquée au texte sur les faces du cube.

Conseils d'utilisation

Comme toujours lorsque cette possibilité est offerte, il est recommandé d'utiliser ce composant en tant que balise HTML.

En effet, comme vous pouvez le constater, bien que ce soit techniquement faisable, créer le composant et son contenu en javascript pur est inutilement lourd et moins lisible.
