Description

Ce composant permet de créer des boîtes de dialogue. Il est possible de l'utiliser au sein d'une interface (popup s'ouvrant au chargement de la vue, comme celle que vous venez de fermer) ou via l'appel aux fonctions RS_Alert() et RS_Confirm().

Démo

La présente boîte de dialogue a été générée directement via le template HTML de la page.

C'est pourquoi elle s'ouvre automatiquement au chargement de la vue.

Utilisation par template HTML

HTML

Utilisation en javascript pur

js

Le constructeur de la classe RS_Dialog offre la possibilité de générer une boîte de dialogue à partir d'un template HTML. Le template HTML dont l'url est passée en paramètre sera utilisé pour générer le contenu de la boîte de dialogue ainsi invoquée.

Conseils d'utilisation

Les fonctions natives alert() et confirm() peuvent être désactivées par l'utilisateur. Outre l'habillage de vos boîtes de dialogue, les fonctions RS_Alert() et RS_Confirm() vous permettront de contourner cette difficulté.
