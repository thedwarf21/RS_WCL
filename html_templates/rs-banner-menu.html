Description

Ce composant est un menu de navigation horizontal hiérarchisé. Il est conçu pour synchroniser son contenu avec une propriété objet de votre controller, via l'attribut rs-model.

Conseils d'utilisation

Comme toujours lorsque cette possibilité est offerte, il est recommandé d'utiliser ce composant en tant que balise HTML.

Il vous faudra sans doute modifier quelques peu le CSS afin de l'adapter à votre page. En effet, le conteneur HTML contenant les éléments de sous-menu est en positionnement absolu. Il vous faudra donc modifier la propriété top du sélecteur div.rs-submenu afin que le conteneur se place juste en-dessous du menu, comme sur la démo.
